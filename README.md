## Stack Technique

- Webpack
- Babel
- Scss

## Exécuter localement

### Installer les dépendances

```bash
npm install
```

### Compiler les fichiers

```bash
npm run dev
```

### Mise en prod

```bash
npm run build
```

Copier le dossier `public` sur le serveur
